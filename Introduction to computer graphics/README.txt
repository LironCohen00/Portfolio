Introduction to Computer Graphics — Ray Tracer
===============================================

A ray tracing renderer implemented in Python as part of a Computer Graphics course.

The implementation renders 3D scenes by simulating the path of light rays,
computing intersections with scene geometry, and applying shading models.

Contents (HW3/):
- hw3.py              — main ray tracing implementation
- helper_classes.py   — scene objects, vectors, rays, and lighting utilities
- Ray Tracing Assignment.ipynb — Jupyter notebook with the full assignment,
                                 explanations, and rendered output

Language: Python
