Logistic Regression
====================

From-scratch implementation of logistic regression with gradient descent
for binary classification.

Topics covered:
- Sigmoid activation function
- Cross-entropy (log loss) cost function
- Gradient descent parameter learning with convergence detection
- Model evaluation on held-out test data

Language: Python
Files:
- hw4.py                        — implementation
- hw4_323929992_806614.ipynb    — notebook with experiments
- training_set.csv / test_set.csv — dataset
