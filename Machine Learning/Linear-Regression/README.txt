Linear Regression
=================

From-scratch implementation of linear regression in Python, covering both
analytical and iterative solutions.

Topics covered:
- Data preprocessing: mean normalization and feature scaling
- Bias trick (augmenting the feature matrix with a ones column)
- Gradient descent optimisation with cost (MSE) tracking
- Closed-form solution via the pseudoinverse (normal equations)
- Forward feature selection for model complexity control
- Polynomial feature generation for non-linear regression

Language: Python
Files: hw1.py (implementation), hw1.ipynb (notebook with experiments)
