Haskell Fundamentals
====================

Introduction to core Haskell concepts through a series of function implementations.

Topics covered:
- Function composition and the (.) operator
- Currying and uncurrying for variable-arity functions
- Number theory utilities: binary conversion (toBinary/fromBinary),
  digit counting, digit rotation
- Divisibility and number properties: sumDivisors, isAbundant

Language: Haskell
File: HW1.hs
