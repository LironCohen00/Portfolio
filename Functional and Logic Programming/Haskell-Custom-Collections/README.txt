Haskell — Custom Map and Set
=============================

Implementation of Map and Set abstract data types from scratch in Haskell,
using comparison-based ordering rather than the standard library.

Topics covered:
- Typeclass definitions and instances
- Equality and ordering abstractions (EqMap, EqSet)
- Building collection types with custom lookup and insertion logic

Language: Haskell
Files: HW4.hs, EqMap.hs, EqSet.hs
