Haskell — Double-Ended Queue (Deque)
=====================================

Implementation of a purely functional double-ended queue (Deque) in Haskell,
supporting efficient insertion and removal from both ends.

Topics covered:
- Purely functional data structure design
- Amortized complexity with lazy lists
- Deque operations: pushFront, pushBack, popFront, popBack

Language: Haskell
Files: HW5.hs, Deque.hs
