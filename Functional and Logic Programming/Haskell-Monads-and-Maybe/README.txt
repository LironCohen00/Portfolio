Haskell — Monads and Maybe/Either
==================================

Exercises focused on monadic programming patterns and error-handling types
in Haskell.

Topics covered:
- Maybe type: concatMaybeMap, fromMaybe, mapMaybe, catMaybes
- Either type: error-propagating computations and error handling
- List transformations using monadic bind
- Chaining operations that may fail

Language: Haskell
File: HW2.hs
