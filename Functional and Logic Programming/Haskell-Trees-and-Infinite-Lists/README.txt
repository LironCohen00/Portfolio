Haskell — Trees and Infinite Lists
====================================

Advanced Haskell implementation of recursive data structures and lazy evaluation.

Topics covered:
- Binary tree serialization and deserialization
- Infinite lists and lazy evaluation
- Custom operators for infinite list processing:
  imap    — map over an infinite list
  ifilter — filter an infinite list
  ifind   — find the first element satisfying a predicate
- Natural number generation using infinite streams

Language: Haskell
File: Main.hs
