HTTP Analysis and Webpage
=========================

Two-part exercise covering the foundations of HTTP and web development.

Part 1 — HTTP Packet Analysis (HTTP-packets.pdf)
  Captured and analyzed raw HTTP request/response packets, examining headers,
  status codes, and the structure of HTTP communication.

Part 2 — HTML Webpage (Webpage.html)
  A hand-coded HTML page featuring a message composition form with fields
  for sender, receiver, subject, and message body. Includes an animated
  heading and embedded images.

Files:
- Webpage.html        — the hand-coded HTML submission
- HTTP-packets.pdf    — packet analysis writeup
- TheoreticalQuestions.pdf — theoretical exercise
- confusedcatjpg.jpg, huh_cat.jpg — images embedded in the webpage
