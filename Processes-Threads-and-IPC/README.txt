Processes, Threads and IPC
==========================

C programming exercises covering core operating systems concepts: process
management, inter-process communication, and synchronisation.

Files:
- Sleep.c — Process forking with sleep-based parent/child synchronisation
- IPC.c — Bidirectional IPC using pipes (parentToChild, childToParent)
- Semaphores.c — Semaphore-based synchronisation using POSIX semaphores (sem_wait/sem_post)
- Threads.c — Multi-threaded program with shared state protected by pthread_mutex
- TheoreticalQuestions.pdf — Theoretical exercise

Language: C
