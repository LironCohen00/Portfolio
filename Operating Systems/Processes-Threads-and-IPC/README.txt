Processes, Threads and IPC
==========================

C programming exercises covering core operating systems concepts: process
management, inter-process communication, and synchronisation.

Exercises:
- Part 1, Q1 — Process forking with sleep-based parent/child synchronisation
- Part 1, Q2 — Bidirectional IPC using pipes (parentToChild, childToParent)
- Part 1, Q3 — Semaphore-based synchronisation using POSIX semaphores (sem_wait/sem_post)
- Part 2, Q2 — Multi-threaded program with shared state protected by pthread_mutex

Files:
- Ex2_part1q1.c  — fork and process synchronisation
- Ex2_part1q2.c  — pipe-based IPC
- Ex2_part1q3.c  — POSIX semaphore synchronisation
- Ex2_part2q2.c  — pthreads and mutex locking
- Instructions.pdf / part3.pdf — exercise specifications

Language: C
